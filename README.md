## System requirements
* **OS:** Windows 7 or higher (projects can also work in Windows 11, but 100% uptime is not guaranteed)

## Build

Compile using [FASM](https://flatassembler.net) or [ASM Visual](https://gri-software.com/en/asmvisual)
